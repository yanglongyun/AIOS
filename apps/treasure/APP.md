# 藏宝阁

## API

- `POST /apps/treasure/upload` - 上传图片
- `POST /apps/treasure/appraise` - AI 鉴宝并入库
- `GET /apps/treasure/list` - 获取藏品列表
- `GET /apps/treasure/detail` - 获取藏品详情
- `GET /apps/treasure/image` - 读取藏品图片
- `POST /apps/treasure/delete` - 删除藏品

## 数据表

- `apps_treasures`
