# AIOS Shared Contributions

This directory archives three historical contributions that were shared into AIOS by other community members.

The files here are source snapshots for reference, not active runtime code. Each contribution folder contains:

- `README.md`: short context and source links.
- `files.txt`: changed files from the original commit.
- `code/`: the actual files as they existed after the original contribution commit, preserving the original path structure.
