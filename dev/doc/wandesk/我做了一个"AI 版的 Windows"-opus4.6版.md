# 我花了大半年,做了一个"AI 版的 Windows"

说出来你可能觉得我疯了。

这一年我一直在想一个特别朴素的问题:为什么我们用了这么久的 AI,桌面还是十年前那个桌面?

我每天打开电脑,Dock 上那一排图标,跟我 2015 年用的没什么区别。Word、浏览器、备忘录、一个记账 App、一个待办 App……每个都是别人替我决定好的样子。我想要个能记录我减脂期每天吃了多少卡路里、还顺便算上我跑步消耗的小工具?对不起,要么下载一个塞满广告和会员墙的,要么自己写代码。

而另一边,AI 已经强到能帮我写整个项目了。可它被关在一个聊天框里。

我越想越觉得不对劲。**对话这个东西,适合表达"我想要什么",但特别不适合"留下来"。** 你不会在一个聊天窗口里记账——记完一关页面,啥都没了。你需要的是一个有界面、有数据、能反复打开的东西。一个 App。

可是做 App 这件事,门槛高得离谱。

于是我开始做一个很轴的东西。我管它叫 Wandesk。

## 它其实就是一台桌面

你打开它,是一个真的桌面——有壁纸,有窗口,能拖能拉。但它没有预装那一堆你用不上的软件。

它只有一个 AI。你跟它说一句话:"帮我做一个记录减脂期饮食的小本子,能算每天的卡路里结余。"

然后它就在你这台机器上,当场把这个 App 给你造出来了。有界面、有数据库、能存东西、明天打开还在。

你想要阅读清单,它给你做阅读清单。你想要个发票生成器,它给你做发票生成器。你想要个跟朋友 AA 分账的工具,它给你做分账工具。这些 App 不是从某个应用商店下载的,是**为你这个人、这个需求,当场长出来的。**

而且它们不是孤岛。这些你造的 App,跟聊天、文件、任务、记忆住在同一个桌面里,AI 在它们之间是记得你的。你昨天告诉它你在减脂,今天它帮你做菜谱 App 的时候就知道要标卡路里。

## 中间我差点放弃过几次

我不想把这篇写成成功学。真实情况是,这东西做起来比我想的难太多。

最折磨的是"怎么让 AI 造出来的东西不丑、不崩"。早期版本,AI 给我糊一个 App 出来,界面像是上个世纪的网页,而且经常造一半就卡死。我改了无数次提示词,推翻重来好几轮架构。

还有一个特别要命的问题:AI 跑在你机器上,它有操作权限,有时候会手贱把自己依赖的服务给关了——然后整个桌面就转圈圈连不上。我盯着那个 "Connecting..." 看了不知道多少回,最后才给它加了个"自动把自己救活"的守护机制。

这种东西没人替你趟坑,只能一个一个自己撞。

## 我想说几句老实话

它现在远不完美。

AI 造复杂 App 还是会翻车,造个小工具靠谱,造个特别大的东西它会力不从心。这是现实,我不藏着。

但有几件事我是认真想清楚了的:

**它完全跑在你本地。** 你的 App、你的数据、你的文件,全在你自己机器上。云端只在你主动调用模型的时候才介入。没有账号,不用注册,我也不收集你那些东西。

**它完全免费。** 你接入你自己的 API Key(Claude、OpenAI、DeepSeek、Kimi,什么都行),用你自己的额度。我没有在中间设卡。

我做它不是为了做个能融资的 SaaS。我就是真的想要这么一台电脑——一台属于我自己的、能随我心意长出软件的电脑。

现在 macOS 和 Windows 都能用了。

如果你也常常有那种"我就想要个小工具,但没人会专门做"的时刻,也许可以试试。做坏了、崩了、哪里别扭,都欢迎来骂我——这玩意儿还在长,你的吐槽就是它下一步往哪长。
