## 一切皆命令

AIOS 的内核只有一个工具——shell。

作为 AI 操作系统的内核，它的职责应该是纯粹的：理解人的意图，转化为命令，交给计算机执行。仅此而已。

Unix 用"一切皆文件"定义了它与硬件世界的关系，一个原语，撑起了几十年的生命力。AIOS 用"一切皆命令"定义了 AI 与计算机的关系。一个 shell，就是那个最小的、完备的原语。

把 coding agent 塞进内核是一种诱惑。但它应该是运行在 AIOS 之上的应用，而不是内核的一部分——就像 vim 运行在 Unix 之上，没有人会把它编译进内核。

有了命令能力，不需要为它构建上下文，那样只会限制它的活动范围，它可以自己创建和查阅笔记，可以自己查询数据库获取对话历史，它甚至可以自己修改自己的回复补充细节，不需要搞什么程序压缩，你需要做的，只是在最初提醒一下它。

这即是大道至简。
