# 应用创建指导

创建新应用前先读目标应用的 `APP.md`（如果是参考现有应用），以及 `memory/architecture/` 中的架构文档。

## APP.md frontmatter

每个应用根目录下必须有 `APP.md`，frontmatter 固定四个字段：

```yaml
---
name: reader
description: 📚 互动式阅读器，支持 AI 生成摘要与解读
backend: apps/reader
database: database/apps/reader.db
---
```

- `name`：小写，无空格，与后端 `name` 字段一致
- `description`：emoji + 简短描述，会注入到 System Prompt
- `backend`：后端目录（相对于项目根）
- `database`：数据库文件路径（无独立数据库则填 `无`）

## 后端结构

### 目录

```
apps/<appname>/
├── APP.md
├── index.js                  应用入口（必须）
├── api/
│   └── index.js              HTTP 路由（必须）
├── service/
│   ├── list.js               一个文件一个操作
│   ├── create.js
│   └── generate.js           AI 任务调用
└── repository/
    ├── client.js             DB 连接（必须）
    ├── init.js               DB 初始化（必须）
    ├── list.js
    └── create.js
```

### 应用入口 index.js

```js
import { handleReaderApi } from './api/index.js';
import { initReaderDatabase } from './repository/init.js';

export default {
  name: 'reader',
  match: (path) => path.startsWith('/apps/reader/'),
  initDb: initReaderDatabase,
  handleApi: handleReaderApi
};
```

必须字段：`name`、`match`、`handleApi`。可选：`initDb`、`initRuntime`。

### 注册到 apps/registry.js

```js
export const appLoaders = [
  () => import('./reader/index.js'),
  () => import('./newapp/index.js'),  // 新增一行
];
```

### API 路由 api/index.js

```js
import { readBody } from '../../../shared/http/readBody.js';
import { json } from '../../../shared/http/json.js';

export const handleReaderApi = async (req, res, path) => {
  if (path === '/apps/reader/list' && req.method === 'GET') {
    return json(res, await list());
  }
  if (path === '/apps/reader/create' && req.method === 'POST') {
    const body = await readBody(req);
    return json(res, await create(body));
  }
  if (path === '/apps/reader/detail' && req.method === 'GET') {
    const url = new URL(req.url, `http://${req.headers.host}`);
    const id = Number(url.searchParams.get('id') || 0);
    return json(res, await detail({ id }));
  }
  return false;  // 未匹配 → 404
};
```

规则：完整路径匹配，不用 `:id`，用查询参数 `?id=xxx`。

### Repository 数据库

```js
// repository/client.js
import { createAppDb } from '../../app_shared/db/createAppDb.js';
export const db = createAppDb('reader.db');

// repository/init.js
import { db } from './client.js';
export const initReaderDatabase = () => {
  db.exec(`CREATE TABLE IF NOT EXISTS items (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT NOT NULL,
    created_at TEXT DEFAULT (datetime('now'))
  )`);
};
```

开发阶段直接删库重建，禁止 `ALTER TABLE`。

### AI 任务调用

```js
// agentTask — 后台异步，支持工具调用
import { agentTaskJson } from '../../app_shared/agentTask.js';
const result = await agentTaskJson({
  app: 'reader',
  title: '任务标题',
  prompt: '提示词'
});

// instantTask — 同步，适合结构化输出
import { instantTaskJson } from '../../app_shared/instantTask.js';
const result = await instantTaskJson({
  app: 'reader',
  title: '标题',
  prompt: '提示词',
  schema: { type: 'object', properties: { summary: { type: 'string' } } }
});
```

## 前端结构

### 目录

```
ui/src/apps/<appname>/
├── index.vue                  主视图（必须）
├── SubComponent.vue           子组件（与主视图同目录）
└── ...
```

### 注册到 ui/src/apps.js

```js
{ id: 'newapp', name: '新应用', icon: '🆕', iconClass: 'icon-newapp',
  load: () => import('./apps/newapp/index.vue') },
```

注册后自动出现在桌面和应用网格中，无需路由配置。

## 修改后的标准动作

### 前端改动

```bash
cd <项目根目录>/ui && npm run build
```

不要在项目根、`ui/src` 或其他目录直接执行构建。构建后刷新浏览器即可看到效果。

### 后端改动（apps/ 或 server/）

重启 Node.js 服务进程即可生效。

> 无多语言机制，所有界面文字直接写中文字面量。
