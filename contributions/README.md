# AIOS Shared Contributions

This directory archives three historical contributions that were shared into AIOS by other community members.

The files here are records, not active runtime code. Each contribution folder contains:

- `README.md`: short context and source links.
- `files.txt`: changed files from the original commit.
- `commit.diff`: textual diff from the original commit. Binary assets are recorded as binary file entries, not duplicated as embedded binary patches.
